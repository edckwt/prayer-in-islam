Prayer In Islam | Your step-by-step easy Prayer Guide


== Description ==

Prayer In Islam aspires to be a unique and simple online guide on Prayer and how to perform it properly. It seeks to teach Muslims how to pray as correctly as Prophet Muhammad did.

Features:

* Getting the content (articles, videos, audios) of Prayerinislam.com.
* Using Json API
* Using Cron Job


PS: All the websites owned by EDC [(E-Da`wah Committee)](http://www.edc.org.kw), For Support click to [Islam.com.kw](http://www.islam.com.kw/).


== Installation ==

* Upload the Plugin to your blog, Activate it.
* Then the content will be stored automatically in your database.
* The plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.
